# Code of Conduct

## Our Commitment
This project aims to maintain a respectful, collaborative, and professional environment for developing a privacy-aware Retrieval-Augmented Generation (RAG) system.

## Expected Behavior
- Be respectful and constructive in discussions
- Provide clear and helpful feedback
- Focus on technical improvement and collaboration
- Follow responsible handling of sensitive data

## Unacceptable Behavior
- Harassment, abusive language, or personal attacks
- Sharing or exposing sensitive or private data
- Submitting malicious or insecure code

## Scope
This applies to all interactions related to this project, including code contributions, issues, and discussions.
